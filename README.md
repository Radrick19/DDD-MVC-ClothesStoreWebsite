# Description
Simple clothes store website made according to ddd pattern

# Features
Functionality:
- Product catalog
- Personal cart
- Product details
- Login/Registration
- Administration(add products, colors, collections, categories)

Tecnologies:
backend
- c# .net 6 ( Asp.net core )

frontend
- HTML5, SCSS, basic js + JQuery

Database - PostgreSQL
